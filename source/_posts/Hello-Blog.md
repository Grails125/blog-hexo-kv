## 关于这个博客

这是一个基于 Hexo 和 Cloudflare Pages 构建的现代化博客系统。

### 功能特性

- 🚀 快速部署
- 📝 Markdown 写作
- 💬 评论系统
- 🎨 现代化主题

### 开始使用

```bash
# 创建新文章
npm run new "文章标题"

# 启动本地预览
npm run dev
```
