# 欢迎来到我的博客

这是一个基于 Hexo 和 Solitude 主题构建的现代化博客系统,部署在 Cloudflare Pages 上。

## 特色功能

### 🚀 高性能

- 静态站点生成,加载速度极快
- Cloudflare CDN 全球加速
- 优化的图片和资源加载

### 💬 评论系统

本博客集成了基于 Cloudflare KV 的评论系统,你可以在文章下方留下你的想法。

### 📊 访问统计

每篇文章都有访问量统计,让你了解文章的受欢迎程度。

### ❤️ 点赞功能

喜欢某篇文章?点个赞让我知道!

### 🔍 全站搜索

使用搜索功能快速找到你感兴趣的内容。

### 📡 RSS 订阅

通过 RSS 订阅,第一时间获取博客更新。

## 技术栈

- **Hexo**: 快速、简洁且高效的博客框架
- **Solitude 主题**: 现代化、美观的主题设计
- **Cloudflare Pages**: 免费的静态站点托管
- **Cloudflare KV**: 键值存储,支持动态功能
- **Cloudflare Workers**: 无服务器函数,处理 API 请求
